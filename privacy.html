Kebijakan Privasi

SaldoSehat membantu pengguna mencatat dan meninjau keuangan pribadi, termasuk pemasukan, pengeluaran, Dompet, target, pinjaman, dan piutang.

Data utama SaldoSehat disimpan di perangkat Anda. Privasi data menjadi tanggung jawab Anda. Notifikasi dapat dibuat privat untuk membantu menjaga privasi.

1. Data yang Anda masukkan

SaldoSehat dapat menyimpan data yang Anda masukkan ke dalam aplikasi, seperti transaksi, nama Dompet, target, catatan pinjaman, catatan piutang, kategori, dan catatan lain yang Anda buat sendiri.

2. Penyimpanan data

Data utama aplikasi disimpan secara lokal di perangkat Anda. Jika aplikasi dihapus, data aplikasi dibersihkan, atau perangkat mengalami kerusakan, data dapat hilang apabila Anda belum membuat backup atau export.

3. Export dan backup

Fitur export dan backup disediakan untuk membantu Anda menyimpan arsip pribadi. Setelah file export atau backup dibuat, keamanan file tersebut menjadi tanggung jawab Anda. Simpan file di tempat yang aman dan jangan membagikannya kepada pihak yang tidak dipercaya.

4. Iklan dan layanan pihak ketiga

Versi gratis SaldoSehat dapat menampilkan iklan. Layanan iklan atau layanan pihak ketiga yang digunakan aplikasi dapat memproses informasi tertentu sesuai dengan kebijakan mereka. Detail teknis dapat berubah mengikuti versi aplikasi dan layanan yang digunakan.

5. Notifikasi

SaldoSehat dapat menggunakan notifikasi untuk pengingat. Anda dapat mengatur notifikasi melalui fitur yang tersedia di aplikasi atau melalui pengaturan perangkat.

6. Tanya SaldoSehat

Tanya SaldoSehat memberikan bantuan berdasarkan template dan data yang tersedia di aplikasi. Fitur ini bukan penasihat keuangan resmi dan tidak menggantikan konsultasi dengan profesional.

7. Keamanan

Kami berusaha menjaga aplikasi tetap sederhana dan aman digunakan. Namun, keamanan perangkat, backup, file export, dan akses ke perangkat tetap menjadi tanggung jawab pengguna.

8. Perubahan kebijakan

Kebijakan Privasi ini dapat diperbarui dari waktu ke waktu untuk menyesuaikan fitur, kebutuhan operasional, atau kebijakan platform.
